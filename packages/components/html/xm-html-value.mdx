# XmHtmlValue
_________
Use it in cases when you want to show HTML that you will receive from API response.

## Selector
`@xm-ngx/components/html-value`

## Widget configuration example
```json
{
  "field": "path.to.your.value.in.stored.value",
  "selector": "@xm-ngx/components/html-value",
  "config": { // config is optional
    "class": "mb-3 d-flex",
    "style": "color: red;"
  }
}
```

# XmHtml
_________
Use it in cases when you want to show HTML that you will pass in config property.

## Selector
`@xm-ngx/components/html`
## Widget configuration example

Case when you need translation:
```json
{
  "selector": "@xm-ngx/components/html",
  "config": {
    "class": "mb-3 d-flex", // class is optional
    "style": "color: red;", // style is optional
    "html": {
      "en": "<div>Amazing HTML is here</div>",
      "uk": "<div>Неймовірний HTML тут</div>"
    }
  }
}
```

Case when you don't need translation:
```json
{
  "selector": "@xm-ngx/components/html",
  "config": {
    "html": "<div>Amazing HTML is here</div>"
  }
}
```
